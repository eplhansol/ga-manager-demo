# 총무 관리 시스템 (GA-Manager)

> 계약 · IT자산 · 보험 · 차량 · 렌탈 · 지식재산권을 한곳에서 관리하는 **사내 총무 관리 시스템**
> Dashboard · 만료 D-DAY 알림 · 월별 비용 현황 · 3단계 권한 관리

🔗 **데모 사이트**: https://eplhansol.github.io/ga-manager-demo/
*(데모는 가상 데이터로 동작하는 프런트엔드 단일 페이지입니다. 실제 운영본은 사내에서 비공개로 운영 중입니다.)*

---

## 📌 배경

총무 업무 전반(계약·자산·보험·지식재산권 등)을 **26개 시트짜리 엑셀 마스터파일** 하나로 관리하고 있었습니다.
- 만료일이 다가오는 항목을 한눈에 보기 어렵고,
- 여러 명이 동시에 편집하면 충돌이 났으며,
- "지금 무엇이 만료 임박인지" 같은 질문에 매번 수작업으로 답해야 했습니다.

이 문제를 풀기 위해, 기존 엑셀의 구조를 그대로 흡수하면서 **검색·알림·권한·통계**를 더한 웹 애플리케이션으로 재설계했습니다.

---

## ✨ 주요 기능

- **대시보드** — 월 고정비(구독·렌탈·보험), 만료 임박/지남 현황, 영역별 카드, **월별 관리 비용 차트**
- **9개 영역 통합 관리** — 계약 / HW·SW 자산 / NAS / 보험 / 차량 / 렌탈 / 특허 / 상표
- **만료 D-DAY 알림** — 임박(D-30·D-7·D-1) 자동 강조, 매일 지정 시각 메일 발송
- **만료 관리 탭** — 만료 지난 항목을 현황판에서 분리해 별도 관리
- **월별 업무사항 보고** — 총무 업무 일지 + 비용 집계
- **3단계 권한(RBAC)** — 관리자(전체 편집) / 뷰어(전체 조회) / 제한된 유저(허용 영역만)
- **전체 검색**, 엑셀 가져오기·내보내기, 변경 이력 기록

## 🛠 기술 스택

| 구분 | 사용 기술 |
|------|-----------|
| Frontend | React, TypeScript, Vite |
| Backend | Node.js, Express, PostgreSQL |
| Infra | Docker Compose, 사내 Synology NAS 배포 |
| 외부 접속 | FortiGate SSL-VPN (사내망 한정 보안 접속) |

- **레지스트리 기반 제네릭 CRUD** — 9개 영역이 하나의 설정(config)으로 동일 로직을 공유
- **공통 레이어(common_item)** — 영역이 달라도 대시보드·검색·알림을 통합 처리

---

## 🖥 데모 안내

- 본 저장소는 **프런트엔드 단일 HTML 데모**입니다. (백엔드/DB 없이 동작)
- 화면의 모든 데이터는 **임의로 생성한 가상 데이터**이며, 실제 회사 정보와 무관합니다.
- 상단 배너에서 **관리자 / 뷰어 / 제한된 유저** 역할을 전환하며 권한별 화면을 비교해볼 수 있습니다.
